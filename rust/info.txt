wasm-pack build --scope k_mj94

Cargo.tomi 설정

[package]
name = "first-wasm"
version = "0.1.0"
authors = ["KimMyeongJun <myeongjun222@naver.com>"] => git에서 사용하는 authors
license = "MIT/Apache-2.0"
repository = "https://github.com/KMJ192/first-wasm"
# See more keys and their definitions at https://doc.rust-lang.org/cargo/reference/manifest.html

[lib]
crate-type = ["cdylib"]

[dependencies]
wasm-bindgen = "0.2"

Cargo.tomi 설정 후
wasm-pack build --scope k_mj94(npm유저네임)
=>
1. Rust코드를 WebAssembly로 컴파일
2. WebAssembly 위에서 wasm-bindgen을 실행하여, WebAssembly를 npm이 
이해할 수 있는 모듈로 감싸는 JavaScript 파일을 생성
3. pkg 디렉터리를 만들고 JavaScript 파일과 WebAssembly 코드를 그 안으로 옮김
4. Cargo.toml을 읽고 동등한 package.json을 생성
5. README.md가 있다면 패키지로 복사

생성된 pkg로 이동 및 나의 npm에 새 패키지 배포 //npm 이메일 인증이 되어있지 않을 경우 403Error발생
cd pkg
npm publish --access=public
